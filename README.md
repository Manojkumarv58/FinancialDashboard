# 💸 Finance Dashboard UI

A modern and responsive **Finance Dashboard** built using React and Redux Toolkit.
This project focuses on clean UI, reusable components, and efficient state management.

> ⚡ This is a frontend-only project using mock data (no backend).

---

## 🚀 Features

### 📊 Dashboard Overview

* Total Balance, Income, Expenses summary cards
* Monthly balance trend (line chart)
* Category-wise expense breakdown (pie chart)

### 💳 Transactions

* Transaction table with date, amount, category, and type
* Search, filter, and sorting functionality
* Empty state handling

### 👥 Role-Based UI (Simulated)

* Switch roles from header dropdown
* Viewer → Read-only access
* Admin → Add and edit transactions

### 📊 Insights

* Highest spending category
* Month-over-month expense comparison

### 🧠 State Management

* Redux Toolkit (`finance` slice)
* Global state for transactions, filters, sorting, and role
* State persistence using `redux-persist`

---

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* Redux Toolkit + React Redux
* redux-persist
* Recharts
